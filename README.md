# SAHAYAK

SAHAYAK is a multi-agent AI system for real estate document intelligence.
It analyzes rental agreements and lease contracts to provide clause-grounded,
safe, and observable AI-generated explanations.

Phase 1 focuses on:
- Document parsing
- Clause-level retrieval
- Hallucination prevention
- LLM observability

The system prioritizes correctness and explainability over generative fluency.
